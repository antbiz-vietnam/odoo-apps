Custom Products Carousel

A clean and responsive product carousel snippet for Odoo Website Builder. Easily drag & drop into any page and customize layout per device.

Key Features

  • ✔ Drag & drop snippet inside Website Builder
  • ✔ Fully responsive on desktop, tablet, mobile
  • ✔ Adjustable rows, columns, and slide speed

Carousel Configuration Options

  • Slider Speed: Time between slide transitions
  • Slider Row (Desktop): Number of rows shown on desktop
  • Slider Row (Mobile): Number of rows shown on mobile
  • Slider Columns (Desktop): Number of columns per slide on desktop
  • Slider Columns (Small Devices): Columns per slide on smaller screens

Installation

  1. Download the module from Odoo Store.
  2. Go to Apps → Upload Module if installing manually.
  3. Ensure the modules website and website_sale are installed.
  4. Click Install.
  5. Open Website → Edit to access the snippet.

How to Use

  1. Open Website → Edit mode.
  2. Go to Blocks → Products.
  3. Drag and drop Custom Products Carousel onto the page.
  4. Click the snippet to open settings and configure layout, rows, columns, and filters.
  5. Save the page to apply changes.
